User Profile Page

Technical Description

The User Profile page was created using **semantic HTML5** and styled with **CSS Flexbox & Grid** for responsiveness. It includes:

  • A **profile card** with an avatar, bio, and contact details.
  • A **volunteering availability table** with checkboxes.
  • A **contact form** with proper labels and ARIA attributes.
  • A **UN Goals section**, displaying icons representing supported initiatives.

Accessibility

The profile page ensures **keyboard navigation support**, **proper contrast ratios**, and **ARIA roles** for assistive technologies.

Challenges and Lessons Learned

One of the main challenges was implementing **SVG interactivity** in the sitemap. I solved it using JavaScript for hover and focus states. Another challenge was ensuring **form accessibility** in the profile page, which required ARIA labeling and proper input focus.

Compliance

All pages follow **WCAG 2.1 accessibility guidelines** and **HTML5 validation** standards. The website contains no copyrighted materials and adheres to **university regulations**.
