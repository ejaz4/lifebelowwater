Splash screen

Technical Description

The splashScreen.html file is designed as a splash screen that plays a background video while displaying the organization's name, mission statement, and a list of contributors. The page automatically redirects to home.html after 4 seconds.

Key features

Structured Content
  • Header: Displays the title “SeaGate” with a tagline
  • Logo: A visually prominent SVG image (/images/logo-filled-negative.svg)
  • Organization Mission Statement: Provides details about the UK-based charity’s goal
  • Group Member List: Acknowledges contributors
  • Animated Loading Indicator
    • Uses three animated circles (div.circle) to indicate page loading

Auto-Redirect

  • meta http-equiv="refresh" content="4; url=home.html" Redirects to home.html after 4 seconds

div.circle

Each div class="circle" represents a small bouncing circle in the loader animation.

How it works

display: flex; in .loader
  • Aligns the three circles horizontally (justify-content: center;)
  • Adds spacing between them (gap: 20px;).
  • Aligns them at the bottom of the container (align-items: flex-end;)
.circle Styles
  • Shape: border-radius: 50%; to make them circular
  • Animation: animation: moveUpAndDown 2s ease infinite; (applies the bouncing effect).
Bouncing Animation (@keyframes moveUpAndDown)
  • 0% and 100%: The circles remain at their original position.
  • 50%: The circles move 50px upwards (translateY(-50px);)
  • The movement creates a bouncing effect.
Animation Delays
  • The 2nd circle starts 0.5 seconds later.
  • The 3rd circle starts 1 second later.
  • This staggered timing creates a wave-like bouncing effect instead of all circles jumping together.

Accessibility

Semantic HTML

  • Uses h1, h2, h3, p, and img appropriately.

Color Contrast & Readability

  • The dark overlay (rgba(0, 0, 0, 0.5)) ensures text remains readable against the video.

Keyboard & Screen Reader Navigation

  • Content is structured logically with clear heading levels.
